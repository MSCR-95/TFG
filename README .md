## Instalación

Instala `uv` si no lo tienes:

```bash
pip install uv
```

Clona el repositorio y entra a la carpeta:

```bash
git clone <url-del-repo>
cd TFG
```

Instala las dependencias:

```bash
uv sync
```

Copia el archivo de configuración:

```bash
cp .env.example .env
```

## Uso

```bash
uv run main.py
```
